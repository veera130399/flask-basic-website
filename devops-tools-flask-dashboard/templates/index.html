Docker

Docker is a tool designed to make it easier to create, deploy, and run applications by using containers.

Kubernetes

Kubernetes is an open-source platform for automating the deployment, scaling, and management of containerized applications.

Jenkins

Jenkins is an open-source automation server used to build, test, and deploy software.

Terraform

Terraform is an open-source Infrastructure as Code (IaC) tool that enables you to define and provision infrastructure using a declarative configuration language.
